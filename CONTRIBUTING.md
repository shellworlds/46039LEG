# Contributing to Tumor Detection Project

## Fork & Clone Workflow

### Step 1: Fork the Repository
1. Go to https://github.com/shellworlds/46039LEG
2. Click the 'Fork' button in the top-right corner
3. Select your account (muskan-dt, mike-aeq, vipul-zius, or dt-uk)

### Step 2: Clone Your Fork
```bash
git clone https://github.com/YOUR-USERNAME/46039LEG.git
cd 46039LEG
```

### Step 3: Add Upstream Remote
```bash
git remote add upstream https://github.com/shellworlds/46039LEG.git
```

### Step 4: Create a Branch for Your Work
```bash
git checkout -b feature/YOUR-REFERENCE-ID
# Example: git checkout -b feature/6039LEGA
```

### Step 5: Make Changes and Commit
```bash
# Make your changes to the code
git add .
git commit -m 'Add: Your feature description'
```

### Step 6: Push to Your Fork
```bash
git push origin feature/YOUR-REFERENCE-ID
```

### Step 7: Create Pull Request
1. Go to your fork on GitHub
2. Click 'Compare & pull request'
3. Reference the issue you're addressing (e.g., Fixes #2)
4. Add your Reference ID in the PR description
